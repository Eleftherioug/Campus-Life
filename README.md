# Campus Life

## Project Description
Campus Life is a simple student-focused web project designed to showcase campus updates, planned features, and event-style content in a clean, responsive interface.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- JSONPlaceholder API

## Features
- Responsive navigation across the main pages
- Semantic HTML structure using `header`, `main`, `section`, and `footer`
- JavaScript button interaction for testing functionality
- Live API content loaded into Bootstrap cards
- Basic responsive styling for desktop and mobile screens

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Launch `index.html` in a web browser.
4. Navigate to `about.html` and `events.html` using the navbar links.
